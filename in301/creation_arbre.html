Création d'arbres

En langage C il n'existe pas de type de variable qui nous permettent de faire des arbres directement, on va alors créer une structure pour mettre en place ces arbres et ainsi manipuler des données. On créer donc un struct dans lequel on met une valeur, c'est le père, il a 2 fils qu'on va appeler fils gauche et fils droit (si c'est une feuille, ces fils sont vides).

On cherche maintenant à faire un arbre plus grand. Pour ça on ajoute un fils droit et un fils gauche au fils droit et gauche de la racine précédente. On a alors sur le schéma : A la racine ayant pour fils droit C et fils gauche B. B est aussi une racine qui a pour fils droit B2 et pour fils gauche B1. C est aussi une racine qui a pour fils gauche C1 et pour fils droit C2. On se rend vite compte que la structure est répétitive. C'est pourquoi on va créer une structure récursive qui va permettre d’appeler la structure dans la structure elle même. De ce fait, on aura une structure telle que A sera une variable int et B et C seront eux des racines d'un nouvel arbre. Ainsi, B sera la racine qui sera une variable int et qui aura pour fils gauche et droit d'autre racine (ici B1 et B2). Pareil pour C qui est la racine int de C1 et C2 qui sont elles même des racines.

 

xxxxxxxxxx
struct noeud {
  int val;          // Valeur de la racine
  struct noeud *fg; // Fils gauche
  struct noeud *fd; // Fils droit
}
​
int main () {
  struct noeud A, B, C;
  A.val = 12;
  B.val = 6;
  C.val = 27;
  A.fg = &B;
  A.fd = &C;
  return 0;
}

On comprend vite qu'avec cette façon de faire, un arbre avec des millions d'entrées va vite devenir très long à faire. On va donc créer des fonctions pour simplifier la création d'arbre.

xxxxxxxxxx
struct noeud {
  int val;
  struct noeud *fg;
  struct noeud *fd;
}
​
typedef struct noeud * ARBRE // on dit qu'on peut écrire struct noeud ou ARBRE c'est la même chose
​
ARBRE creer_feuille (int val) {
  struct noeud *n:                  // on peut aussi écrire ARBRE n
  n = malloc(sizeof(struct noeud)); // On réserve de la mémoire pour créer une variable qui sera réutilisé et pourra être pointé                                     // (avec des *n ou des n->)
  if (n == NULL) exit(2);           // On retourne un code erreur, 2 pour qu'à la compilation, on sait quelle erreur c'est
  n->val = val;                     // ou bien (*n).val = val;
  n->fg = NULL;                     // ou bien (*n).fg = NULL;
  n->fd = NULL;                     // ou bien (*n).fd = NULL;
  
  return n;                         // On retourne l'arbre n
}
​
// fusion de 2 arbres
ARBRE fusion (int val, ARBRE G, ARBRE D) {
  ARBRE A;
  A = creer_feuille(val);
  A->fg = G;                        // Le fils gauche de A devient l'arbre gauche
  A->fd = D;                        // Le fils droit de A devient l'arbre droit
  return A;                         // On renvoit l'arbre A
}
​
// On veut afficher l'arbre pour visualiser et voir si ce qu'on a fait fonctionne
// La fonction est récursive parce qu'on veux afficher tous les noeuds 
void affiche_arbre (ARBRE A) {
  if (A == NULL) return;            // Si l'arbre est vide alors on ne rien faire
  affiche_arbre(A->fg);
  printf(" %d",A->val);
  affiche_arbre(A->fd);
}
​
// On libère récursivement les noeuds de l'arbre pour tout libérer et tout vider
void free_arbre(ARBRE A) {
  if (A==NULL) return;              // Si l'arbre est vide alors on n'a rien à faire
  free_arbre(A->fg);                // on libère le sous arbre gauche 
  free_arbre(A->fd);                // On libère le sous arbre droit
  free(A);
}
​
int main () {
  ARBRE Y, Z, X;
  Y = creer_feuille(12);
  Z = creer_feuille (34);
  X = fusion (78, X, Y);
  affiche_arbre(X);
  printf("\n");
  
  Z = creer_noeud(1);
  X = fusion(2,NULL,Z);
  Y = creer_feuille;
  Z = fusion(3,X,Y);
  affiche_arbre(Z);
  printf("\n");
