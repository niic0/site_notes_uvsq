Rappel

Chaîne hamiltonienne : Chaîne qui passe par tous les sommets une fois. Cycle hamiltonien : Chemin hamiltonien qui est un cycle. Chaîne Eulérien : Chaîne qui passe une fois par chaque arrête. Une chaîne Eulérienne existe si 2 sommets sont de degrés impairs et tous les autres de degré pairs. Cycle Eulérien : Chaîne Eulérienne qui revient au sommet de départ. Un cycle Eulérien existe si tous les sommets sont de degré pair.
